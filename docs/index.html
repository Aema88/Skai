Система керування мінітеплицею на базі IoT з використанням хмарної платформи AWS

IoT-Based Control System for a Mini Greenhouse Using Amazon Web Services Cloud Platform

Короткий опис: Розробка інтелектуальної системи моніторингу та керування кліматом у мінітеплиці за допомогою сенсорів, мікроконтролерів та хмарних сервісів AWS.

Ключові слова: IoT, теплиця, AWS, сенсори, автоматизація, управління кліматом

Мета дослідження

Створити масштабовану систему моніторингу та управління мікрокліматом мінітеплиці з використанням хмарної платформи Amazon Web Services та технологій Інтернету речей.

Основні завдання

  • Проєктування та побудова апаратної частини системи
  • Розробка програмного забезпечення для збирання даних з сенсорів
  • Інтеграція з AWS (IoT Core, Lambda, DynamoDB)
  • Створення інтерфейсу для користувача
  • Тестування і оцінка системи

Очікувані результати

Автоматизована, доступна з будь-якого пристрою система управління теплицею з можливістю масштабування на більші проєкти.
